### 显示/隐藏 隐藏文件

- 快捷键的方式：`shift+cmmand+.`

- 终端的方式：

  ```shell
  ls -al
  ```


### 移动文件

`command + c` => `command + option + v`

### 复制内容到便利贴

`command + shift + Y`

### 隐藏当前窗口

`command + H`

### 隐藏除了当前窗口之外的其他窗口

`command + option + H`

### 关闭所有窗口

`command + option + W`
